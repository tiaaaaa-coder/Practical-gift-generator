# Practical Gift Generator 🎁

## Project Description

The Practical Gift Generator is a web application that helps users find useful gift ideas based on their preferences, budget, and occasion.

## Features

* Suggests practical gift ideas
* Allows users to choose a budget
* Filters gifts based on category
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
* (Optional) Node.js for backend

## Software Architecture

The system follows a **3-layer architecture**:

1. Presentation Layer (Frontend)
2. Application Layer (Backend logic)
3. Data Layer (Gift data storage)

## How to Run the Project

1. Clone the repository
2. Open `frontend/index.html`
3. Use the app in your browser
